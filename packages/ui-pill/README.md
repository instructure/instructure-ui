## ui-pill

Use Pills to communicate concise status with an identifiable color and supportive text.

### Components
The `ui-pill` package contains the following:
- [Pill](#Pill)


### Installation

```sh
yarn add @instructure/ui-pill
```

### Usage
```jsx
---
example: false
---
  <Pill
    color="info"
    margin="x-small"
  >
    Draft
  </Pill>
```
