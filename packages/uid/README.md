## uid

[![npm][npm]][npm-url]&nbsp;
[![MIT License][license-badge]][license]&nbsp;
[![Code of Conduct][coc-badge]][coc]

A unique (CSS-safe) id generator made by Instructure Inc.

### Installation

```sh
yarn add @instructure/uid
```

### Usage

```javascript
import uid from '@instructure/uid'

const id = uid()
```
