[![MIT License][license-badge]][license]

### Installation

```sh
npm install @instructure/ui-icons
```

### Usage

To use the React components:

```js
---
type: code
---
import { IconAddLine } from '@instructure/ui-icons'

const MyComponent = () => {
  return <IconAdd />
}
```
